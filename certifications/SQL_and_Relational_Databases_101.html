SQL and Relational Databases 101

SQL

Overview

Awarded by IBM in February 2025, this certification demonstrates my proficiency in SQL and relational database concepts. The course covered: Relational Model & Constraints – Understanding data models, relationships, and integrity constraints. SQL Fundamentals – Writing queries using SELECT, INSERT, UPDATE, DELETE, and CREATE TABLE statements. Advanced SQL Techniques – String patterns, result set sorting, and grouping. Joins & Multi-Table Queries – Efficiently combining data from multiple tables using INNER and OUTER JOIN operations.

  • Writing efficient SQL queries for data retrieval and manipulation
  • Implementing advanced SQL techniques for optimized querying
  • Hands-on experience with real-world database scenarios
  • Strengthened foundation for database management and optimization
